# Diagrams

This folder contains selected public diagrams from Livingtelligence.

They are included as standalone PNG files so they can be viewed directly in GitHub.

## Included Diagrams

- `livingtelligence-flow.png`: shows the movement from signals to phenomena, observation, organization, and institutional evolution.
- `custodian-what-it-does.png`: shows Custodian as one agentic component inside the broader Livingtelligence architecture.

## Public Scope

These diagrams are public-facing explanatory artifacts.

They do not describe the full internal implementation, data model, private registries, credentials, or operational configuration of Custodian.
